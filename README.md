# Funny Quote Generator

The Funny Quote Generator is a web application that fetches random funny quotes from the API Ninjas Funny Quotes API and displays them on the webpage.

## Features

- Generates random funny quotes from the API Ninjas Funny Quotes API.
- Displays the generated quote and its author on the webpage.
- Simple and intuitive user interface.

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/your-username/funny-quote-generator.git
    ```

2. Navigate to the project directory:

    ```bash
    cd funny-quote-generator
    ```

3. Open the `index.html` file in your web browser.

## Usage

- Click the "Generate Quote" button to fetch a new random funny quote from the API.
- The generated quote and its author will be displayed on the webpage.

## API Key

To use the Funny Quote Generator, you need to sign up for an API key from API Ninjas. Once you have the API key, replace the placeholder `"Your-API-Key-Goes-Here"` in the `script.js` file with your actual API key.
